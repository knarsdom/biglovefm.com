#บอยรถบ้าน จัดหนัก #แจกหมียักษ์ กับแฟนๆ #BigLoveFM…#สัปดาห์ที่3

กิจกรรม / 2 ปี ago

#บอยรถบ้าน จัดหนัก #แจกหมียักษ์ กับแฟนๆ #BigLoveFM
Lucky Bear Lucky You ร่วมสนุกได้ทุกสัปดาห์ ลุ้นตุ๊กตาหมียักษ์ทุกเดือนกับเกมส์ “#เข้าเป้าเขย่าโชคกับบอยรถบ้าน”
สัปดาห์ที่ 3 จันทร์ที่ 20 มีนาคม 60 ภาพที่ 3 ภารกิจที่ 3 แล้วนะคะ ใครทำภารกิจภาพแรกไม่ทัน ตามดูรายละเอียดที่ www.biglovefm.com นะจ๊ะ พร้อมแล้วลุย!!!!
วิธีการ 2 สเต๊ป คือต้องเป็นเพื่อนกะบอยรถบ้านก่อน โดย
1. Add Line เพิ่มเป็นเพื่อนกับ #บอยรถบ้าน ID Line : @smn1970y


2.จากนั้น ส่งทักทายไปในไลน์ #บอยรถบ้าน พิมพ์ชื่อเล่น พร้อมใส่รหัส BL เช่น #น้ำBL // #จูนBL // #แบงค์BL (แค่นี้เราก็ได้สนิทสนมกันแว้วววววว)
3.เล่มเกมส์ ในเฟสบุ๊ค FanPage : BigLove FM เกมส์ #เข้าเป้าเขย่าโชคกับบอยรถบ้าน
4.เมื่อภารกิจสำเร็จให้แคปเจอร์ภาพนั้นเก็บไว้
5.ทำครบภารกิจ 4 สัปดาห์ 4 ภาพ ให้ส่งภาพทั้ง 4 กลับมาที่ไลน์บอยรถบ้าน (ภาพจะถูกอัพโหลดทุกวันจันทร์เวลา09.00 น ที่เฟสบุ๊ค FanPage : BigLove FM)
6.ผู้ที่ทำภารกิจสำเร็จ และส่งมาก่อนเป็นท่านแรกรับตุ๊กตาหมีนำโชค ไซด์ยักษ์ 1รางวัล
ติดตามรายละเอียดเพิ่มเติมที่ www.biglovefm.com
#หมียักษ์ตัวใหญ่ใครๆก็อยากได้
